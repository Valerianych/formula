# Встроенная база данных для Vercel

В проект добавлен вариант хранения данных прямо внутри сайта. Это не требует запускать SQLite на компьютере и не требует отдельного сервера базы данных.

## Как это работает

Данные Формулы-1 лежат в файле:

```text
src/data/f1VercelData.ts
```

Vercel API Functions из папки `api/` читают эти данные и отдают их сайту через те же маршруты, которые уже использует React-интерфейс.

## Какие API есть

```text
/api/sessions
/api/session-data
/api/driver-laps
/api/standings
/api/results
/api/analyze
/api/db-status
```

## Как проверить на Vercel

После деплоя открыть:

```text
https://адрес-сайта.vercel.app/api/db-status
```

Если всё нормально, вернётся JSON с количеством сессий, гонок и пилотов.

## Как объяснять на защите

В проекте используется встроенная локальная база данных в виде структурированного TypeScript-модуля. Данные о сезонах, гонках, пилотах, кругах, погоде и победителях хранятся внутри проекта. На Vercel они выдаются через serverless API, поэтому сайт не зависит от отдельного сервера базы данных и может работать сразу после деплоя.

Проще сказать так: данные уже лежат внутри сайта, а API просто достаёт нужную информацию и показывает её пользователю.
